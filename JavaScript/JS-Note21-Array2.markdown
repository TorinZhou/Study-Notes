## Simple Array Methods

- Without Mutate Original Array

  - Slice

    ```javascript
    let arr = ["a", "b", "c", "d", "e"];
    [...arr]; //[abcde]
    arr.slice(); // [abcde]
    arr.slice(1); // [bcde]
    arr.slice(-1); // [e]
    arr; // [abcde]
    ```
